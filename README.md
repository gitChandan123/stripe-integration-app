## 1. Prerequisites

Before starting, make sure you have the following installed:

- Python 3.x
- Django 4.x
- Stripe Account (for API keys)
- Virtual Environment (`venv`)

If you don't have a Stripe account, you can create one at [Stripe](https://stripe.com/).
